# dftpl rules in YAML format

The example of YAML format for dftpl rule is shown below:

```
- description: ""
- analyzer_category: ""
- test_event:
    - type: ""
    - evidence: ""
- high_level_event:
    - pattern: ""
    - type: ""
    - description: ""
    - keys:
        - key1: ""
        - key2: ""
- reasoning:
    - description: ""
```
